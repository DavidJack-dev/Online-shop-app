
# Travel reservation service app


> 미리 예약된 여행 상품을 판매 해주는 중개 서비스 사이트입니다.
> 불가피하게 여행을 못가게 되어서, 이 앱을 통해서 판매자는 제품을 올린 후 구매자는 카트에 담은 후 결제 하는 시스템입니다.
